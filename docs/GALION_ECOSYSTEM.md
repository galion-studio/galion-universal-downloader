## âš¡ Serverless Deployment Guide

### RunPod Deployment

```bash
# Clone the repository
git clone https://github.com/galion-studio/runpod-universal-downloader
cd runpod-universal-downloader

# Install dependencies
npm install

# Set environment variables
export PORT=3000
export NODE_ENV=production
export DOWNLOADS_DIR=/tmp/downloads

# Start in serverless mode
npm run start:serverless
```

### Docker Deployment

```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

### Environment Variables

```env
# Server
PORT=3000
NODE_ENV=production

# Downloads (use /tmp for serverless)
DOWNLOADS_DIR=/tmp/downloads

# API Keys (optional)
CIVITAI_API_KEY=
GITHUB_TOKEN=
YOUTUBE_API_KEY=
TELEGRAM_BOT_TOKEN=
```
